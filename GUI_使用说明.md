# FastMCP Content Factory GUI 使用说明

## 🚀 快速启动

### 方法1: 智能启动（推荐）
```bash
python start_gui_smart.py
```
自动检测系统深色/浅色主题，选择最合适的界面

### 方法2: 直接运行
```bash
python content_generator_gui_dark.py
```

## 📋 功能特性

### ✅ 多平台内容生成
- **微信公众号**: 2000-5000字深度分析文章
- **小红书**: 500-1500字生活分享内容
- **B站**: 1000-2500字知识科普视频脚本
- **抖音**: 200-500字短视频脚本

### ✅ 智能功能
- **反审查模式**: 自动模型切换（Qwen3 → Claude），突破内容限制
- **Tavily 真实搜索**: 研究阶段使用实时网络数据，非模拟结果
- **结构化输出**: 标题+正文一次生成，质量稳定
- **质量评分**: 实时评估内容质量（CASE 框架合规度）
- **结果保存**: JSON/TXT 格式导出

## 🎯 使用流程

### 1. 输入内容主题
- 在「内容主题」框中输入主题（建议具体描述）
- 例：「小米 SU7 2024年销量与市场竞争分析」

### 2. 选择目标平台
- 勾选需要生成内容的平台（可多选）

### 3. 设置内容类型
- **📰 文章**: 常规图文内容
- **🎬 视频脚本**: 含镜头指导和时间轴

### 4. 开始生成
- 点击「🚀 开始生成内容」或按 **Ctrl+G**
- 实时显示生成进度

### 5. 查看与保存结果
- 分标签页显示各平台内容
- 显示质量分、字数、信息密度等指标
- 一键复制到剪贴板 / 保存为文件

## ⚙️ 环境配置

### 依赖安装
```bash
uv sync
```

Python 版本要求：**3.10+**

### API 密钥配置
编辑 `.env` 文件：
```env
OPENAI_API_KEY=your_yunwu_key
OPENAI_API_BASE=https://yunwu.ai/v1
TAVILY_API_KEY=your_tavily_key
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+G | 开始生成 |
| Ctrl+S | 保存结果 |
| Ctrl+R | 重置界面 |
| Ctrl+N | 选择新主题 |
| Ctrl+T | 切换主题模式 |
| F5 | 清空结果 |
| F1 | 显示帮助 |

## 🛡️ 故障排除

**Q: GUI 启动失败**
A: 确认 Python 3.10+ 和 tkinter 已安装，运行 `uv sync` 安装依赖

**Q: 内容生成失败**
A: 检查 `.env` 中 API 密钥是否正确，网络是否正常

**Q: Tavily 搜索无结果**
A: 确认 `TAVILY_API_KEY` 已配置，系统会自动降级使用 LLM 内置知识

## 💡 使用技巧

1. **主题描述越具体效果越好**：「苹果Vision Pro 2024年销售数据与用户反馈」> 「苹果产品」
2. **开启反审查模式**：处理商业分析、政策解读等话题
3. **多平台组合**：同时勾选多个平台，一次生成全套内容
4. **查看质量指标**：关注「信息密度」和「CASE合规度」评分
