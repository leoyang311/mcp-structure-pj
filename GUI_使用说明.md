# FastMCP Content Factory GUI 使用说明

## 🚀 快速启动

### 方法1: 一键启动（推荐）
```bash
python start_gui.py
```

### 方法2: 直接运行
```bash
python content_generator_gui.py
```

## 📋 功能特性

### ✅ 100%可用性保证
- **兼容性**: 支持Python 3.8+，使用内置tkinter库
- **容错性**: 自动检测依赖，缺失时使用模拟模式
- **稳定性**: 多线程设计，UI不会卡顿

### ✅ 多平台内容生成
- **微信公众号**: 2000-5000字深度分析文章
- **小红书**: 500-1500字生活分享内容
- **B站**: 1000-2500字知识科普视频脚本
- **抖音**: 200-500字短视频脚本

### ✅ 智能功能
- **反审查模式**: 自动模型切换，突破内容限制
- **质量评分**: 实时评估内容质量
- **批量生成**: 同时为多个平台生成内容
- **结果保存**: JSON/TXT格式导出

## 🎯 使用流程

### 1. 输入内容主题
- 在"内容主题"框中输入你要创作的主题
- 支持中英文，建议使用具体描述

### 2. 选择目标平台
- 勾选需要生成内容的平台
- 可以同时选择多个平台
- 每个平台有独特的内容风格

### 3. 设置内容类型
- **📰 文章**: 常规文案内容
- **🎬 视频脚本**: 包含镜头指导和时间轴
- **🖼️ 图文混排**: 结合视觉元素的内容

### 4. 配置高级选项
- **反审查模式**: 处理敏感话题时开启
- **质量模式**: standard/high/premium三种质量

### 5. 开始生成
- 点击"🚀 开始生成内容"按钮
- 实时显示生成进度
- 支持中途取消

### 6. 查看结果
- 分标签页显示各平台内容
- 显示生成信息（模型、质量分、字数等）
- 一键复制到剪贴板

## 📁 文件说明

### 核心文件
- `content_generator_gui.py` - 主GUI程序

### 配置文件
- `.env` - API密钥配置
- `.env.example` - 配置模板

## ⚙️ 环境配置

### 必需环境
```bash
Python 3.8+
tkinter (通常已内置)
```

### API密钥配置（可选）
创建 `.env` 文件：
```env
# OpenAI API密钥 (用于真实AI生成)
OPENAI_API_KEY=your_openai_key

# Tavily API密钥 (用于研究功能)  
TAVILY_API_KEY=your_tavily_key
```

### 完整依赖安装
```bash
pip install openai python-dotenv requests asyncio
```

## 🛡️ 故障排除

### Q: GUI启动失败
**A:** 运行 `python start_gui.py`，会自动检查并修复环境问题

### Q: 内容生成失败
**A:** 检查以下情况：
1. 网络连接是否正常
2. API密钥是否正确配置
3. 是否开启了反审查模式

### Q: 没有API密钥怎么办？
**A:** GUI支持模拟模式，会生成演示内容供测试使用

### Q: 界面显示异常
**A:** 确保Python版本 >= 3.8，tkinter正常安装

## 💡 使用技巧

### 高质量内容生成
1. **主题描述具体化**: "小米汽车2024年市场分析" 比 "汽车分析" 效果更好
2. **开启反审查模式**: 处理商业、政治敏感话题
3. **选择高质量模式**: premium质量生成更专业内容
4. **多平台组合**: 同时生成多个平台内容，节省时间

### 内容优化建议
1. **微信公众号**: 适合深度分析、行业报告
2. **小红书**: 适合生活分享、产品评测
3. **B站**: 适合知识科普、教程解析
4. **抖音**: 适合快节奏、情感共鸣内容

### 批量操作
1. 预设多个主题，逐个生成
2. 利用保存功能建立内容库
3. 复制功能快速分发到各平台
