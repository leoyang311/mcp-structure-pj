# GUI 更新说明

## ✨ 当前版本特性（v2.1）

### 1. **深色主题 GUI**
- 专属深色主题：背景 `#2b2b2b`，文字 `#ffffff`，高对比度
- 强调色：蓝色 `#0078d4`，系统原生感
- 字体：SF Pro Display (macOS) / Segoe UI (Windows)

### 2. **智能主题检测**
- `start_gui_smart.py` 自动检测系统深色/浅色模式
- 运行时可通过 **Ctrl+T** 一键切换，状态保持不丢失

### 3. **多次连续使用**
- **重置按钮**：完全重置到初始状态，无需重启
- **清空按钮**：仅清空结果，保留设置
- **完成对话框**：生成后自动弹出操作选项（保存/继续/重置）
- **热门主题选择器**：15个预设主题快速切换

### 4. **真实 API 集成**（v2.1 新增）
- Tavily 真实网络搜索替换模拟数据
- 结构化输出（Pydantic）替代手动 JSON 解析
- AsyncOpenAI 客户端支持高并发生成

### 5. **快捷键支持**
| 快捷键 | 功能 |
|--------|------|
| Ctrl+G | 开始生成 |
| Ctrl+S | 保存结果 |
| Ctrl+R | 重置界面 |
| Ctrl+N | 选择新主题 |
| Ctrl+T | 切换主题模式 |
| F5 | 清空结果 |
| F1 | 显示帮助 |

## 🚀 启动方式

### 推荐：智能启动器
```bash
python start_gui_smart.py
```

### 直接启动
```bash
python content_generator_gui_dark.py
```

```
mcp-structure-pj/

```

## 🔧 故障排除

| 问题 | 解决方法 |
|------|---------|
| 文字不可见 | 已默认使用深色主题，无需切换 |
| 生成失败 | 检查 `.env` API 密钥和网络连接 |
| 快捷键无效 | 确保主窗口获得焦点 |
| 保存失败 | 检查当前目录写入权限 |
| 界面卡顿 | 关闭其他大型程序，使用重置功能清理内存 |
