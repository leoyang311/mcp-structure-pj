## MiniMax Prompt方案

### 一、MiniMax核心特性
- **生成时长**: 6秒
- **优势领域**: 人物动作、光影效果、材质表现
- **分辨率**: 1280×720 (高清)
- **帧率**: 25fps

### 二、MiniMax提示词结构

```
[镜头类型] + [主体描述] + [动作] + [环境] + [光线] + [风格] + [质量标签]
```

### 四、MiniMax提示词优化技巧

#### 1. 时间控制标记
```
[0-2秒]: 缓慢开始，建立场景
[2-4秒]: 主要动作展开
[4-6秒]: 高潮时刻，视觉冲击
```

#### 2. 质量增强词汇
- **清晰度**: 8K, 超高清, 锐利细节, 纹理清晰
- **光影**: 体积光, 光线追踪, 全局照明, 次表面散射
- **色彩**: 10-bit色深, 广色域, 专业调色, 电影级LUT
- **渲染**: 物理渲染, 真实材质, PBR贴图, 光线追踪

#### 3. 避免使用的词汇
- 快速剪切、频繁切换
- 复杂变形、形态转换
- 多人复杂互动
- 极限运动、高速碰撞

## Sora Prompt方案

### 一、Sora核心特性（预期）
- **生成时长**: 最长60秒
- **核心优势**: 物理准确、长时连贯、复杂叙事
- **分辨率**: 最高1920×1080
- **特色**: 3D空间理解、持续性记忆

### 二、Sora提示词架构

```
[时间线] + [叙事结构] + [角色/物体追踪] + [物理描述] + [空间一致性] + [摄影技术]
```

### 三、Sora长视频提示词模板

#### 1. 完整叙事类（60秒）

##### 案例一：品牌故事
```
[0-10秒] 晨光中的咖啡种植园，航拍缓慢下降到咖啡树
[10-20秒] 农民手工采摘咖啡豆，特写豆子从红色到深褐色
[20-30秒] 转场到烘焙工厂，咖啡豆在烘焙机中翻滚
[30-40秒] 咖啡师研磨咖啡，蒸汽机工作，制作espresso
[40-50秒] 顾客端起咖啡杯，满足的表情，晨光透过窗户
[50-60秒] 镜头拉远，咖啡馆全景，品牌logo淡入

物理真实性：
- 咖啡豆翻滚遵循重力
- 蒸汽上升的流体动力学
- 液体倒入杯中的真实流动

视觉连贯性：
- 咖啡豆作为视觉线索贯穿全片
- 色调从绿色自然过渡到棕色温暖
- 光线从户外自然光到室内暖光
```

##### 案例二：科技产品发布
```
[0-15秒] 设计师在数字画板上勾勒产品草图，线条逐渐成型
[15-25秒] 3D模型生成，各个零件组装动画，展示内部结构
[25-35秒] 原型机在实验室测试，数据在屏幕上实时显示
[35-45秒] 量产线上机器人精密组装，品质检测过程
[45-55秒] 用户开箱瞬间，表情从好奇到惊喜
[55-60秒] 产品在各种场景中使用，快速剪辑蒙太奇

技术规格：
- 保持CAD线条到实物的视觉延续
- 装配动画物理准确
- 机器人运动路径真实
```

## 提示词优化策略

### 一、通用优化原则

#### 1. 清晰性原则
- 使用具体数值而非模糊描述
- 明确时间、速度、距离
- 详细描述材质和纹理

#### 2. 一致性原则
- 统一视觉风格描述
- 保持色调和光线连贯
- 维持物理规则统一

#### 3. 层次性原则
- 先整体后局部
- 先主体后背景
- 先动作后细节

### 二、模型特定优化

#### MiniMax优化重点
1. **单一焦点**: 6秒内聚焦一个核心动作
2. **视觉冲击**: 强调画面美感和构图
3. **光影效果**: 充分利用其光线处理优势
4. **材质表现**: 详细描述表面质感

#### Sora优化重点
1. **叙事完整**: 充分利用60秒讲述完整故事
2. **物理准确**: 详细描述物理参数
3. **空间连贯**: 强调3D空间的一致性
4. **角色追踪**: 保持人物和物体的持续性

### 三、提示词测试框架

```
测试维度：
1. 生成成功率
2. 视觉质量评分
3. 动作流畅度
4. 物理真实性
5. 提示词匹配度

评分标准（1-10分）：
- 8-10分：生产可用
- 6-7分：需要微调
- 6分以下：重新设计
```

## 质量检查清单

### 提示词质量检查

#### 基础检查项
- [ ] 是否包含镜头类型
- [ ] 是否明确主体对象
- [ ] 是否描述动作/变化
- [ ] 是否说明环境背景

- [ ] 时间标记是否清晰

- [ ] 视觉风格是否统一
